== Short Description ==

Customize the WordPress login page with your own logo and settings.

== Description ==

WPall CustomLogin allows you to customize the WordPress login page with your own logo and settings. You can upload a custom logo, set its size, and make the login logo link to your homepage.

== Installation ==

1. Upload the `custom-login` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Custom Login' menu in the admin panel to configure your custom login settings.

== Frequently Asked Questions ==

= How do I upload a custom logo? =

You can upload a custom logo by navigating to the 'Custom Login' settings page in the WordPress admin and using the upload button.

= Can I adjust the size of the logo? =

Yes, you can set the height and width of the logo in the settings page. The height is in pixels and the width is in percentage.
